<h2>🛠 How to Customize</h2>
<p>Edit the <code>CONFIG</code> object at the bottom of <code>index.html</code>:</p>
<pre>
const CONFIG = {
    heroPrompt: "your AI image prompt",
    notifThumbPrompt: "your thumbnail prompt",
    brandName: "YOUR.BRAND",
    avatarEmoji: "🚀",
    hintText: "your CTA text",
    badgeCount: "5",
    cardA: { label: "Revenue", glyph: "$12K", name: "Monthly", pct: "+45%" },
    cardB: { label: "Users", name: "Active", pct: "2.4K" },
    barHeights: [40,65,85,100,80,55,35]
};
</pre>
